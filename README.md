# Reboot'eux — démo

Site vitrine premium pour Luc Dacquin, énergéticien · rebouteux · coupeur de feu.
Cabinet de soins basé à 30 km de St-Julien-en-Genevois (France / Suisse romande).

## Tech

Site statique HTML / CSS / JS avec React + Babel via CDN. Vidéos cinématiques en background.
Aucun build nécessaire — déployable tel quel sur GitHub Pages.

## Voir le site

[https://rebooteux-demo.github.io](https://rebooteux-demo.github.io) (à venir)

## Structure

- `index.html` — page principale
- `app.jsx` — composants React (Hero, Acts, Luc, Contact, Footer…)
- `components.jsx` — composants réutilisables (VideoBg, Cursor, Nav, Reveal…)
- `styles.css` — palette + typographie + design tokens
- `videos/` — fonds vidéo cinématiques (Hero, Feu, Eau, Terre, Souffle, Étoiles)
- `assets/` — logo Reboot'eux + portrait de Luc
